# Rock Paper Scissors Player - freeCodeCamp Challenge

Repositorio para el desafío de piedra, papel o tijera de freeCodeCamp con solución optimizada que vence a los cuatro bots del reto.

## Archivos incluidos

- RPS.py: función principal player (IA con reconocimiento de patrones).
- main.py: ejecuta partidas automáticas y permite interactuar contra los bots.
- RPS_game.py: lógica del juego y los bots, sin modificar.
- test_module.py: pruebas automáticas del reto.

## Uso rápido

```bash
python main.py
```

Descomenta líneas para jugar manualmente o lanzar los tests.
